# AI-Powered MSP Ticket Triage Engine 🤖

An automated incident parsing utility built using Python and the Google GenAI SDK. This application simulates an automated Level 2 Cloud Support workflow, processing production-level server errors into structural triage reports.

### 🚀 Key Capabilities
- **Automated Root-Cause Analysis (RCA):** Breaks down complex log faults instantly.
- **SLA Protection:** Speeds up Mean Time to Repair (MTTR) by isolating issues before a human operator intervenes.
- **Client Document Generation:** Auto-drafts ready-to-send technical email copy.
